## 说明
### 1、介绍
为了将勇芯科技云端数据共享给第三方使用，简化对接流程，特将部分接口提供出去(目前接口和sdk提供的升级服务保持一致，如果需要其他接口，请咨询商务，根据可行性，进行开放)。具体使用流程是：  
第一，我司分发对应的key给贵公司  
第二，贵司通过这个key和用户的唯一id(用户名，身份证，手机号等)，申请token  
第三，携带token，去申请接口  
第四，部分接口获取数据的前提，是先调用上传历史数据的接口  

- 接口基准地址：`https://ali.bravechip.cn/api/`
- 认证方式：Bearer Token（JWT）
- 数据格式：JSON
  
请求头：
```http
Authorization: Bearer <token>
Content-Type: application/json
Accept: application/json
```
响应样式：
```http
{
  "count": 0,
  "message": "string",
  "result": {},
  "status": 0
}
```
状态码说明：
```java
状态码	说明
200	请求成功
400	参数错误
401	未授权
403	禁止访问
404	资源不存在
500	服务器错误
```


### 1、申请key
合作方可以联系我们，提供贵公司的名称，我们分配调用服务的key
### 2、申请token
根据key，和使用sdk的用户的唯一id(用户名，身份证，手机号等)，就可以申请token，token会自动保存在本地，不需要用户保存，尽量在合适的时候，调用这个接口，刷新一下Token，防止被踢或者Token过期  
#### 请求URL ####
POST app/createTokenForThirdPartCompany
#### 请求参数 ####
```json
{
  "apiKey": "申请的key",
  "username": "用户唯一id"
}
```
#### 响应示例 ####
```json
{
  "result": {
    "token": "eyJhbGciOiJIUzUxMiJ9.........ZyDQ"
  },
  "message": "成功",
  "count": 0,
  "status": 200
}
```
将获取到的token放在请求头里，后续接口都会校验

    public Long parseLongFromBytes(byte[] data, int startIndex, int length) {
        try {
            // 字节数大于4,从起始索引开始向后处理4个字节,其余超出部分丢弃
            final int len = length > 4 ? 4 : length;
            byte[] tmp = new byte[len];
            System.arraycopy(data, startIndex, tmp, 0, len);
            return bitOperator.fourBytesToLong(tmp);
        } catch (Exception e) {
            e.printStackTrace();
            return 0L;
        }
    }
 public int parseIntFromBytes(byte[] data, int startIndex, int length) {
        return this.parseIntFromBytes(data, startIndex, length, 0);
    }

    private int parseIntFromBytes(byte[] data, int startIndex, int length, int defaultVal) {
        try {
            // 字节数大于4,从起始索引开始向后处理4个字节,其余超出部分丢弃
            final int len = length > 4 ? 4 : length;
            byte[] tmp = new byte[len];
            System.arraycopy(data, startIndex, tmp, 0, len);
            return bitOperator.byteToInteger(tmp);
        } catch (Exception e) {
            e.printStackTrace();
            return defaultVal;
        }
    }

    /**
     * 把一个4位的数组转化位整形
     *
     * @param value
     * @return
     * @throws Exception
     */
    public long fourBytesToLong(byte[] value){
        // 小端模式：第一个字节是最低有效字节(LSB)
        return (value[0] & 0xFFL) |
                ((value[1] & 0xFFL) << 8) |
                ((value[2] & 0xFFL) << 16) |
                ((value[3] & 0xFFL) << 24);
    }

   /**
     * 把byte[]转化位整形,通常为指令用
     *
     * @param value
     * @return
     * @throws Exception
     */
    public int byteToInteger(byte[] value) {
        int result = 0;
        for (int i = value.length - 1; i >= 0; i--) { // 从最后一个字节开始处理
            result = (result << 8) | (value[i] & 0xFF);
        }
        return result;
    }
