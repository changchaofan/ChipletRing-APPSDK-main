##### 1. Introduction
This SDK enables engineers to quickly understand and integrate BraveChip's smart ring solutions.

It encapsulates low-power Bluetooth (Bluetooth standard), BraveChip smart ring communication protocols (proprietary), and BraveChip's physiological algorithms.

This SDK needs to be integrated with a real smart ring for deployment and functional testing.

There are two versions of this SDK: Android and iOS.

##### 2. Directory Overview
./Android/example: Example design on the Android side based on the SDK.

./Android/library: Android SDK library.

./Android/source: Partially open-source code on the Android side, used with the SDK.

./IOS/example: Example design on the iOS side based on the SDK.

./IOS/library: iOS SDK library.

./IOS/source: Partially open-source code on the iOS side, used with the SDK.

./Doc/: Important documentation about low-power Bluetooth technology and the smart ring solution.
